## Style Guidelines to Emphasize in Prompts

For consistency across all visualizations, emphasize these elements:

- **Dimensional Depth:** Always include foreground, midground, and background elements
- **Quantum Uncertainty:** Elements that appear to exist in multiple states/positions simultaneously
- **Spatial Navigation:** Grid lines, perspective guides, or navigational elements suggesting movement through dimensional space
- **Tech-Organic Balance:** Combine precise geometric forms with organic energy flows
- **Limited Color Evolution:** Maintain the core blue/purple palette but show an evolution from simpler to more complex color relationships
- **Binary/Code Integration:** Include subtle binary or code elements as texture rather than focus
- **Light as Information:** Use light sources and glow effects to highlight information flows
