Overview

The Quantum-Spatial Design System provides a comprehensive, M4-optimized design system with automatic Cloudflare Worker integration for Framer projects. This demo showcases the components, layouts, and design tokens available in the system.

Multiple Design States

Cloudflare Integration

Integration Code

To integrate the Quantum-Spatial Design System with Framer and Cloudflare, add this code to your project:

// Import the design system provider import { QuantumSpatialProvider, Button, Card } from "quantum-spatial-design-system"; export function App() { return ( <QuantumSpatialProvider autoDetect={true} initialState="quantum" workerUrl="https://design-system.9bitstudios.io" > <YourContent /> </QuantumSpatialProvider> ); } function YourContent() { const { tokens, designState, setDesignState } = useQuantumSpatial(); return ( <div> <Card title="Welcome" elevated={true}> <p>Welcome to the Quantum-Spatial Design System.</p> <Button text="Learn More" variant="primary" /> </Card> </div> ); }
