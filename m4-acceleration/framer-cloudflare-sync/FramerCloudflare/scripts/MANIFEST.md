 # Script Manifest

## Validated Core Scripts
- `deploy-quantum-spatial-worker.sh` (05/11/2025) - Deploys worker to Cloudflare
- `framer-quantum-spatial-sync.js` (05/11/2025) - Syncs Quantum-Spatial Design System with Framer

## Pipeline Components
- `token-generator-script.js` (05/19/2025) - Generates design tokens in various formats
- `framer-sync-script.js` (05/19/2025) - Handles Framer API synchronization for design tokens and components
- `component-batch-processor.js` - Processes components in batch mode
- `svg-processor.js` - Transforms SVGs to quantum-spatial style

- `animation-component-template.js` - Template for creating animated components
- `ar-component-template.js` - Template for augmented reality components
- `brand-component-template.js` - Template for brand-specific components
- `component-builder-script.js` - General component builder utility
- `dashboard-component-template.js` - Template for dashboard UI components
- `quantum-spatial-automation-script.js` - Automation script for quantum-spatial design system
- `svg-to-tsx.js` - Converts SVG files to React TSX components
- `tsx-to-svg-script.js` - Converts React TSX components to SVG files

## Configuration Files
- `framer-sync-config.js` - Configuration for Framer API integration
