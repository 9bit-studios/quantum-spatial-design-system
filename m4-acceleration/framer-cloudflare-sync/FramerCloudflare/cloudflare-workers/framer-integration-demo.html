Quantum-Spatial Design System - Framer Integration Demo

Checking connection to design system...

This demonstration shows how the Quantum-Spatial Design System components would appear in Framer after connecting using the steps outlined in the Framer Integration Guide.

Connection to Design System

// Connect Framer to the design system
framer sync setup --url https://simple-quantum-design-system.rnrb2ynd5z.workers.dev/framer-components

// Pull components into Framer
framer sync pull

Quantum Pixel Component

The foundational building block of the Quantum-Spatial Design System, with support for different quantum states.

Heritage

Transitional

Quantum

Superposition

// In Framer
import { QuantumPixel } from "design-system"

export function Component() {
  return (
    <div style={{ display: "flex", gap: "20px" }}>
      <QuantumPixel state="heritage" size={64} />
      <QuantumPixel state="transitional" size={64} />
      <QuantumPixel state="quantum" size={64} />
      <QuantumPixel state="superposition" size={64} />
    </div>
  )
}
