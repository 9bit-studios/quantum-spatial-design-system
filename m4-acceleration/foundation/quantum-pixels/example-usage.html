Quantum-Spatial Pixel System

This page demonstrates the core SVG templates for the Quantum-Spatial pixel system, optimized for M4 processors.

Pixel States

The Quantum-Spatial system features four primary pixel states that represent different energy conditions.

Materialized

Fully materialized state with solid appearance

Partial

Partially materialized state with semi-transparency

Energy

Pure energy state with glow and particle effects

Superposition

Multiple overlapping states in quantum superposition

Implementation Example

Here's how to use the pixel generator in your code:

// Import the pixel generator
const { generateQuantumPixel } = require('./pixel-generator.js');

// Generate a materialized pixel
const materializedPixel = generateQuantumPixel({
  size: 32,
  state: 'materialized',
  color: '#331F4A', // Dimensional Eggplant
  accentColor: '#5AC8FA', // Subtle Cyan
  borderRadius: 2
});

// Generate a superposition pixel with custom colors
const superpositionPixel = generateQuantumPixel({
  size: 48,
  state: 'superposition',
  color: '#0D0D15', // Midnight Richness
  accentColor: '#6A3093', // Quantum Violet
  secondaryAccentColor: '#FF2D55', // Quantum Pulse Pink
  borderRadius: 3
});

// Generate the entire system with all variations
const pixelSystem = generateQuantumPixelSystem();
