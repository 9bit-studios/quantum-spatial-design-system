Key Properties

  • type: 'background', 'interface', or 'feature'
  • density: 'fine' (8px), 'medium' (16px), or 'coarse' (32px)
  • state: 'heritage', 'transitional', or 'quantum'
  • enablePerspectiveShift: Enables mouse/device-based perspective shift
  • enableSecondaryGrid: Adds diagonal grid lines for depth
  • enableFalloff: Fades grid lines based on distance from center
