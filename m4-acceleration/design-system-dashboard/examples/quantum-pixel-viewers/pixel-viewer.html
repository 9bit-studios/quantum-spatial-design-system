Quantum-Spatial Pixel System

Interactive viewer for the core pixel states in the Quantum-Spatial design system, optimized for M4 processors.

Quantum Pixel States

The quantum-spatial pixel system includes four primary states representing different energy and materialization conditions.

Materialized

Fully materialized state with solid appearance and subtle accent highlight.

Partial

Partially materialized state with translucency and energy particles.

Energy

Pure energy state with dynamic particles, glow effects, and ripple animations.

Superposition

Multiple states simultaneously existing with dimensional displacement effects.

Implementation Guide

The Quantum-Spatial Pixel System is implemented through parameterized SVG templates that can be used to generate all needed variants.

Using the Pixel Generator

// Import the pixel generator
const { generateQuantumPixel } = require('./pixel-generator.js');

// Generate a materialized pixel
const materializedPixel = generateQuantumPixel({
  size: 32,
  state: 'materialized',
  color: '#331F4A', // Dimensional Eggplant
  accentColor: '#5AC8FA', // Subtle Cyan
  borderRadius: 2
});

// Generate an energy pixel with custom colors
const energyPixel = generateQuantumPixel({
  size: 48,
  state: 'energy',
  color: '#2C5F2D', // Heritage Green
  accentColor: '#3DFF74', // Heritage Pixel Green
  borderRadius: 3
});

// Generate a superposition pixel
const superpositionPixel = generateQuantumPixel({
  size: 64,
  state: 'superposition',
  color: '#0D0D15', // Midnight Richness
  accentColor: '#6A3093', // Quantum Violet
  secondaryAccentColor: '#FF2D55', // Quantum Pulse Pink
  borderRadius: 4
});

// Generate the entire system with all variations
const pixelSystem = generateQuantumPixelSystem();
