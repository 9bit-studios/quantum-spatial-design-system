  • Use directional lighting to create natural color variation rather than harsh gradients

Overview

The Quantum Pixel System forms the foundation of our design language, representing our evolution from gaming heritage to technological innovation. Each pixel exists in multiple states, creating a dynamic visual system that adapts to different contexts.

Pixel States

[Pixel State Visualization]

Implementation

Implement Quantum Pixels using the following guidelines:

  • Always include at least 3 different pixel states in any composition
  • Maintain minimum spacing between pixels (at least 50% of the pixel size)
  • Ensure quantum pixels never create recognizable patterns from classic games
  • When animating, transition between states using fade, particle, or dimensional shift effects

Code Example

// CSS implementation example
.quantum-pixel {
  --size: 16px;
  width: var(--size);
  height: var(--size);
  position: relative;
  transition: all 0.6s cubic-bezier(0.33, 0.0, 0.67, 1.0);
}

.quantum-pixel.heritage {
  background-color: var(--heritage-green);
  box-shadow: 0 0 5px rgba(61, 255, 116, 0.3);
}

.quantum-pixel.transitional {
  background-color: var(--subtle-cyan);
  box-shadow: 0 0 8px rgba(90, 200, 250, 0.5);
  transform: perspective(500px) rotateX(15deg);
}

.quantum-pixel.quantum {
  background-color: var(--quantum-violet);
  box-shadow: 0 0 12px rgba(106, 48, 147, 0.7);
  transform: perspective(500px) rotateX(30deg) rotateY(15deg);
}
